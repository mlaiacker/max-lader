# max-lader
DIY Battery Charger, 12V Input, 0-25V Output, 5.5A, using AVR

## Charging
Type | Cells
----|----
Lipo | 1-6
LiFe | 1-7
NIMH/NICA| 1-16
PB | 1-6 (2-12V)
Programmable Power Supply | 0-25V 0-5.5A
